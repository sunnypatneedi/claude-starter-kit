## Core Learning Principles

### The Learning Pyramid

```
RETENTION RATES (average)
├── Lecture: 5%
├── Reading: 10%
├── Audio-Visual: 20%
├── Demonstration: 30%
├── Discussion: 50%
├── Practice: 75%
└── Teaching Others: 90%
```

### Spaced Repetition

```
REVIEW SCHEDULE
Day 1: Learn
Day 2: First review
Day 4: Second review
Day 7: Third review
Day 14: Fourth review
Day 30: Fifth review
Day 60: Sixth review
```

### Deliberate Practice

```
1. SPECIFIC GOAL: Focus on one sub-skill
2. FULL ATTENTION: No distractions
3. FEEDBACK: Immediate and accurate
4. STRETCH: Just beyond current ability
5. REPETITION: Drill weak points
```

## Study Techniques

### Active Recall

Instead of re-reading:

1. Close the book
2. Write down everything you remember
3. Check what you missed
4. Focus on gaps

### Feynman Technique

1. **Choose concept** to learn
2. **Explain it** like teaching a child
3. **Identify gaps** where explanation breaks down
4. **Review and simplify** until clear

### Interleaving

Don't practice one thing repeatedly. Mix:

- Topic A, Topic B, Topic A, Topic C, Topic B
- Forces brain to retrieve and discriminate

### Elaboration

Connect new information to existing knowledge:

- "How does this relate to X?"
- "Why does this work this way?"
- "When would I use this?"

## Skill Acquisition Framework

### Stage 1: Cognitive (1-2 weeks)

- Understand what you're trying to do
- Break down into components
- Watch experts, read guides
- **Goal**: Mental model of the skill

### Stage 2: Associative (weeks to months)

- Practice with feedback
- Identify and fix errors
- Build muscle memory
- **Goal**: Consistent execution

### Stage 3: Autonomous (months to years)

- Skill becomes automatic
- Can focus on strategy/creativity
- Continue deliberate practice on weak areas
- **Goal**: Effortless performance

## Memory Techniques

### Chunking

Group information into meaningful units:

- Phone: 415-555-1234 (not 4155551234)
- Concepts: Group related ideas

### Mnemonics

- **Acronyms**: ROY G BIV (colors)
- **Method of Loci**: Place items in familiar locations
- **Stories**: Weave facts into narrative

### Dual Coding

Combine verbal + visual:

- Draw diagrams
- Create mind maps
- Sketch while listening
