## Core Frameworks

### The Habit Loop

```
CUE → CRAVING → RESPONSE → REWARD

1. CUE: Trigger that initiates behavior
2. CRAVING: Motivation/desire
3. RESPONSE: The actual habit
4. REWARD: The benefit that satisfies the craving
```

### The Four Laws of Behavior Change

```
TO BUILD A GOOD HABIT:
1. Make it OBVIOUS (Cue)
2. Make it ATTRACTIVE (Craving)
3. Make it EASY (Response)
4. Make it SATISFYING (Reward)

TO BREAK A BAD HABIT:
1. Make it INVISIBLE
2. Make it UNATTRACTIVE
3. Make it DIFFICULT
4. Make it UNSATISFYING
```

### Habit Stacking

```
"After I [CURRENT HABIT], I will [NEW HABIT]."

Examples:
- After I pour my morning coffee, I will write in my journal for 2 minutes.
- After I sit down at my desk, I will write down my top 3 priorities.
- After I finish dinner, I will put my phone in another room.
```

## Environment Design

### Friction Reduction

```
GOOD HABITS → Remove friction
- Put gym clothes by bed
- Keep healthy snacks visible
- Set out tomorrow's outfit tonight
- Pre-load apps you want to use

BAD HABITS → Add friction
- Unplug TV after each use
- Delete apps from phone
- Use website blockers
- Leave wallet in car
```

### Visual Cues

```
- Habit tracker on wall
- Sticky notes at decision points
- Physical objects as reminders
- Phone wallpaper with goal
```

## Streak Management

### Building Momentum

```
Week 1: Focus on showing up (even 2 minutes)
Week 2: Establish the pattern
Week 3: Increase duration/intensity slightly
Week 4: It starts feeling normal
Week 5+: Protect the streak
```

### Handling Missed Days

```
RULE: Never miss twice.

If you miss once:
1. Don't catastrophize
2. Get back on track immediately
3. Do the minimum viable version
4. Reflect on what caused the miss
```

## Behavior Change Strategies

### Start Small

```
Want to: Exercise daily
Start with: Put on workout shoes

Want to: Meditate 20 minutes
Start with: One deep breath

Want to: Read more
Start with: Read one page
```

### Temptation Bundling

```
Pair something you NEED to do with something you WANT to do:
- Listen to podcasts only while exercising
- Watch favorite show only while on treadmill
- Eat dessert only at nice restaurants
```

### Identity-Based Habits

```
Instead of: "I want to run a marathon"
Think: "I am a runner"

Instead of: "I want to lose weight"
Think: "I am someone who doesn't miss workouts"

Ask: "What would a [identity] person do?"
```

## Output Format

When designing a habit:

1. **Clarity**: Specific, measurable behavior
2. **Implementation**: Cue, routine, reward
3. **Friction audit**: What could get in the way
4. **Tracking method**: How to measure progress
5. **Recovery plan**: What to do if you slip
